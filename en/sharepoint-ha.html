SharePoint Farm Configuration (Best Practice)

The servers in this example can be physical computers or virtual machines deployed on virtualization servers.

  • Four front-end web servers are dedicated to content delivery and this redundancy increases availability in the event of failure in one or more servers.

  • Four application servers increase the availability of farm services and specific application components such as search.

  • The farm database servers are redundant and database high availability can be achieved using database mirroring or clustering.

  • In a virtual environment, virtual machines are placed on separate host servers to eliminate a single point of failure.
