A highly available Microsoft Exchange Server architecture should contain the following elements:

  • At least two Mailbox Servers to host user mailboxes.
  • servers (Mailbox Server) must be configured as members of a Database Availability Group (DAG) to provide redundancy for Exchange databases.
  • A transport server (Exchange Edge) to manage the delivery of email messages.
  • A gateway server (Exchange Edge or other solution) for spam and virus filtering to secure the incoming mail flow.

The servers in this example can be physical computers or virtual machines deployed on virtualization servers.
